`type エイリアス`＝「型別別名」的意思。
エイリアス＝ alias（別名）。

---

## 定義

使用 `type` 為某個型別取一個名稱，讓後續程式碼引用更簡潔。

```ts
type UserId = number;
```

`UserId` 就是 `number` 的別名。

---

## 用途

1. 簡化複雜型別

```ts
type Result = string | number | null;
```

2. 統一定義資料結構

```ts
type User = {
  id: number;
  name: string;
};
```

3. 搭配字面量與 union

```ts
type Status = "loading" | "success" | "error";
```

---

## 本質

`type` 不產生新功能，僅建立「型別的命名」，方便管理與維護。
