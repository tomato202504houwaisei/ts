在 TypeScript 中，`enum` 用來建立「具名的常數集合」。核心功能是把一組固定且有限的值，用語意化的名稱封裝成一個型別，讓程式碼在可讀性與結構上都更明確。

---

## 1. 基本概念

`enum`＝列舉型別。
用途：定義一組固定、不會任意改變的選項。

例如角色類型、狀態、方向、權限。

---

## 2. 基本範例

```ts
enum Role {
  Student,
  Instructor,
}

let r: Role = Role.Student;
```

TS 自動從 0 開始給值：

- `Role.Student` = 0
- `Role.Instructor` = 1

---

## 3. 自訂值

```ts
enum HttpStatus {
  OK = 200,
  NotFound = 404,
}
```

---

## 4. 字串 enum（實務常用）

避免預設數字造成混淆。

```ts
enum UserRole {
  Student = "student",
  Instructor = "instructor",
}
```

---

## 5. 使用場景

- **角色／權限**
- **系統狀態（Loading/Success/Error）**
- **固定分類（性別、按鈕種類）**
- **HTTP 狀態碼**

---

## 6. `enum` 與物件的差異

使用 enum 的目的：

- 讓值形成「型別本身」，能被 TypeScript 嚴格檢查。
- 物件只能當資料容器，無法產生新的型別。

範例比較：

```ts
enum Direction {
  Up,
  Down,
}

let d: Direction = Direction.Up; // 型別安全

// 若用物件
const Dir = { Up: 0, Down: 1 };
let d2 = 123; // 無法限制
```

---

## 7. JavaScript 輸出結果（編譯後）

```js
var Role = {
  0: "Student",
  1: "Instructor",
  Student: 0,
  Instructor: 1,
};
```

---

## 8. 核心一句話總結

`enum` 用來建立「固定選項的型別」，讓程式碼更安全、更結構化、更好維護。
