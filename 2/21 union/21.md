`union`（聯合型別）用來表示「變數可以是多種型別中的一種」。
核心功能：限定可接受的型別集合，而不是完全開放。

---

## 定義方式

```ts
let value: string | number;
```

代表 `value` 可以是 `string` 或 `number`。

---

## 行為示例

```ts
let x: string | number;

x = "hello"; // OK
x = 123; // OK
x = true; // NG（不在 union 中）
```

---

## 搭配 literal type 使用

常見於狀態、模式、動作類型。

```ts
type Status = "loading" | "success" | "error";

let s: Status = "loading";
```

---

## 搭配 function 提升安全性

```ts
function printId(id: string | number) {
  if (typeof id === "string") {
    console.log(id.toUpperCase());
  } else {
    console.log(id + 1);
  }
}
```

TS 會強制你做型別縮小，避免錯誤操作。

---

## 使用場景

- API 回傳可能是多種型別
- 字串常數集合（類似 enum）
- 表示有限選項的條件
- 表示允許多格式的輸入

---

## 核心一句話

`union`＝允許多種型別，但必須在指定集合內。
